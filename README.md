# sedona layout

![travis-ci](https://api.travis-ci.org/imciflam/sedona-layout.svg?branch=master)

travel website's adaptive layout

### for dev purposes:
1. install packages
```npm i```
2. run dev server
```gulp start```
3. open index.html

### for deploying:
1. create a prod-ready build
```gulp serve```
2. push it to gh-pages
```npx gulp deploy```
